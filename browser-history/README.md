# Browser History

Low-level design practice: **browser-history**

## Run

From the repo root:

```bash
./gradlew :browser-history:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.browserhistory` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

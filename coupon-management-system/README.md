# Coupon Management System

Low-level design practice: **coupon-management-system**

## Run

From the repo root:

```bash
./gradlew :coupon-management-system:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.couponmanagementsystem` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

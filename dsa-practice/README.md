# Dsa Practice

Low-level design practice: **dsa-practice**

## Run

From the repo root:

```bash
./gradlew :dsa-practice:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.dsapractice` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

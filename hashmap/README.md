# Hashmap

Low-level design practice: **hashmap**

## Run

From the repo root:

```bash
./gradlew :hashmap:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.hashmap` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

# In Memory Jigsaw Puzzle

Low-level design practice: **in-memory-jigsaw-puzzle**

## Run

From the repo root:

```bash
./gradlew :in-memory-jigsaw-puzzle:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.inmemoryjigsawpuzzle` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

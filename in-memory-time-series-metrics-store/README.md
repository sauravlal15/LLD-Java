# In Memory Time Series Metrics Store

Low-level design practice: **in-memory-time-series-metrics-store**

## Run

From the repo root:

```bash
./gradlew :in-memory-time-series-metrics-store:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.inmemorytimeseriesmetricsstore` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

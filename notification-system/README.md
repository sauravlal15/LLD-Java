# Notification System

Low-level design practice: **notification-system**

## Run

From the repo root:

```bash
./gradlew :notification-system:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.notificationsystem` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

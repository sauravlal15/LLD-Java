# Pub Sub System

Low-level design practice: **pub-sub-system**

## Run

From the repo root:

```bash
./gradlew :pub-sub-system:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.pubsubsystem` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.

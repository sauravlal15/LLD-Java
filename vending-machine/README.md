# Vending Machine

Low-level design practice: **vending-machine**

## Run

From the repo root:

```bash
./gradlew :vending-machine:run
```

Or from this folder:

```bash
../gradlew run
```

## Package

`com.saurav.lld.vendingmachine` — keep all types for this problem in this package (or subpackages under it) so they never clash with other problems.
